# 202323090-MohammedAlfaraj-assignment3

## Project Description
This is my Assignment 3 portfolio website, which is a continuation of Assignment 1 and 2 and fixing.
The goal of this assignment is to improve the portfolio website by fixing previous issues with Assignment 2, adding advanced functionality, stronger JavaScript logic , API integration, better state management, and performance improvements while also keeping the design responsive, user-friendly, and clear for different devices.


## Features
- Resposive Layout
- Light and Dark Mode Theme Toggle
- Saved theme preference using localStorage
- Greeting message based on the time of day
- Project Search
- Project Filter by Type
- Project Sort by Title
- GitHub API Integration on display repositories
- Contact Form Validation 
- Success and error messages for form feedback
- Visitor name saved using localStorage
- Site viewing counter
- Hover effects on cards and transisions
- Lazy loading for images


## How to Run Locally
1. Clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` using a browser or a local server.
